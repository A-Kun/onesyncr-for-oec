User Stories:

  1. As Professor Han Reinhart, I want to check for updates in other astronomical catalogues daily.
  2. As Professor Han Reinhart, I want to receive an email to notify me if there is any updates.
  3. As Professor Han Reinhart, I want the email comes with a merge request so that once accepted, it merges the updates to my catalogue automatically.
  4. As Professor Han Reinhart, I want to be able to see a list of changes from previous merges.
  5. As Professor Han Reinhart, I want to reject a merge request if I think the source is not trustworthy.
  6. As Professor Han Reinhart, I want to be able to accept the merge request if I think the source is trustworthy.
  7. As Professor Han Reinhart, I want all of the information from other catalogues to come with attributions and references.
  8. As Betty Cho, I want to be able to restore my data to the last version with a click of a button.
  9. As Betty Cho, I want to see a list of updates of the day with a click of a button.
  10. As Professor Han Reinhart and Betty Cho, I want two columns to display and compare the data when there are two or more updates from different catalogues for the same planet.
  11. As Professor Han Reinhart, I want to be able to change the email addresses that receive the notifications.
  12. As Professor Han Reinhart and Betty Cho, I need a setting to configure what time I want notifications to be sent to my email.
  13. As Professor Han Reinhart, I want the program to send me an email notification when the program fails to connect to any of the databases during daily checks.
  14. As John Doe, I want to be able to edit the information from the source manually before merging to the professor’s database.
